Настройка за 2 минуты

Выберите ваше устройство, чтобы получить инструкцию

01

Скачайте приложение

Мы рекомендуем v2rayNG. Это бесплатно, надежно и поддерживает протоколы Reality.

Скачать из Google Play
02

Скопируйте ключ

Зайдите в наш бот или личный кабинет и нажмите кнопку "Скопировать ключ". Он начинается на vless://

vless://96b0147... (пример ключа)
03

Подключитесь

  • 1. Откройте v2rayNG.
  • 2. Нажмите (+) сверху справа → "Импорт из буфера обмена".
  • 3. Выберите появившийся сервер и нажмите круглую кнопку (V) снизу.
01

Установите V2Box

Лучшее приложение для iPhone. Полная поддержка наших протоколов.

Скачать из AppStore
02

Добавьте ключ

Скопируйте ключ доступа в боте. Откройте V2Box — он сам предложит добавить ключ из буфера обмена.

Если не предложил: нажмите "Import V2ray URL"

03

Запуск

Нажмите переключатель Connect. При первом запуске iPhone спросит разрешение на добавление VPN конфигурации — нажмите "Разрешить" и введите пароль от телефона.

01

Hiddify Next

Самый современный и красивый клиент для Windows. Работает там, где другие не могут.

Скачать с GitHub

Выбирайте файл с окончанием .exe (Setup)

02

Активация

1. Скопируйте ключ доступа.
2. Откройте Hiddify.
3. Нажмите большой Плюс (+) или "Добавить из буфера".
4. Нажмите большую кнопку по центру для подключения.
